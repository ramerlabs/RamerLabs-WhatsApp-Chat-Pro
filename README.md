# RamerLabs WhatsApp Chat Pro

![RamerLabs WhatsApp Chat Pro](https://github.com/ramerlabs/RamerLabs-WhatsApp-Chat-Pro/raw/main/assets/images/product.jpg)

Smart WhatsApp floating chat widget for WordPress — route visitors by page, respect business hours, rotate agents, track clicks, and show GDPR consent before opening WhatsApp.

## Features

- **Page routing** — Send sales, support, or billing traffic to the right team based on URL, page ID, post type, or WooCommerce context
- **Business hours** — Online/offline status with “we’ll reply in X hours” messaging
- **Pre-filled messages** — Context-aware templates for products, posts, and pages
- **Agent round-robin** — Distribute chats across multiple WhatsApp numbers per department
- **Click analytics** — UTM tracking and click log in wp-admin
- **GDPR consent** — Optional consent modal before opening WhatsApp
- **Inline buttons** — Gutenberg block, Elementor widget, and `[rlwc_button]` shortcode
- **RamerLabs license SDK** — Updates and activation via ramerlabs.com

## Installation

1. Upload the `ramerlabs-whatsapp-chat-pro` folder to `/wp-content/plugins/`
2. Activate **RamerLabs WhatsApp Chat Pro** in WordPress
3. Go to **WhatsApp Chat → Agents** and add phone numbers (country code, no `+`)
4. Configure routing rules and widget settings
5. Activate your license under **WhatsApp Chat → License**

## Requirements

- WordPress 6.0+
- PHP 7.4+
- WooCommerce optional (for product/shop routing)

## Admin menus

| Menu | Purpose |
|------|---------|
| Dashboard | Overview and quick setup |
| Settings | Widget, hours, templates, GDPR, UTM |
| Agents | Team members and phone numbers |
| Routing | Page-to-department rules |
| Analytics | Click stats and recent activity |
| License | Activate your RamerLabs license key |
